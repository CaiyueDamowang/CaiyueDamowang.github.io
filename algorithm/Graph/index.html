# 图和图算法

非线性的数据结构

图是一种（包含若干个节点），每个节点可以连接 0 个或多个元素

两个节点相连的部分称为边（edge）。节点也被称作顶点（vertice）。一个顶点的 度（degree）是指与该顶点相连的边的条数。

圈是至少有一条边的路径，且路径的第一个顶点和最后一个顶点相同。无论是有向图还是无向图，只要是没有重复边或重复顶点的圈,就是一个简单图。除了第一个和最后一个顶点意外，路径的其他顶点有重复的圈称为平凡圈.

无权图的基础算法：

  • 广度优先遍历

  • 深度优先遍历
