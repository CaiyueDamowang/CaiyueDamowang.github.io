# 链表

链表（ linked list ）： 其中的各对象按线性顺序排列。链表的顺序是由各个对象里的指针决定。

双向链表( doubly linked list ): 每一个元素都是一个带有关键字key和两个指针next、prev

链表的搜索： 可能要搜索整个链表（因为是根据头结点的指针逐一搜索）

链表的插入/删除： 运行时间: O(1)

# 思路

设置快慢指针

遍历链表

反转结点（保存前后结点）
