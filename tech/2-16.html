# webpack基础概念

  • entry: 单入口和多入口 构建依赖树的入口
  • output: 打包后输出的文件 多入口用占位符作为文件名称 [name].js
  • loader: 将遇到的依赖打包成想要的文件

比如 .vue

use: 'vue-loader'
1

比如 .less

use: ['less-loader', 'style-loader', 'css-loader']
1

  • plugins: 各种插件会帮助打包成想要的效果，比如创建html承载输出的脚本

  • dev-server: 不刷新浏览器，实现改变文档

  • dev-middleware: 将文档输出给服务器

原理:bundle.js文件被注入HMR runtime HMR runtime 和 服务器建立websocket链接，更新文件的变化
