# 改变对象的原型

解决了缺少对象在实例化后改变原型的标准方法

let person = { name: 'David' }

let dog = { name: 'wang!' }

let friend = Object.creat(person)

friend.name // 'David'

// 更改原型
Object.setPrototypeOf(friend, dog)

friend.name // 'wang!'
